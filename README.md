# Assignment-3

1. Recursive Factorial (Task1.py)
Description: A program that calculates the factorial of a given integer using a recursive approach.

Key Logic: The function factorial(n) breaks the problem down by multiplying n by the factorial of n-1 until it reaches the base case of 1.

2. Math Operations (task2.py)
Description: A script that takes a numerical input and performs multiple mathematical operations using the built-in math library.

Functions Used:

math.sqrt(): Calculates the square root.

math.log(): Calculates the natural logarithm.

math.sin(): Calculates the sine of the value.


How to Run
Ensure you have Python 3.x installed.

Navigate to the Assignment_3 directory.

Run the programs using the following commands:

    python Task1.py
    python task2.py
